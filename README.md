# SENTINEL

Real-Time Social Media Sentiment Analytics Platform

## Architecture

RSS Feed
↓
Kafka Producer
↓
Kafka Consumer
↓
PostgreSQL
↓
RoBERTa Sentiment Analysis
↓
FastAPI
↓
Streamlit Dashboard

## Features

- Real-time RSS ingestion
- Kafka streaming
- Sentiment Analysis
- FastAPI Backend
- PostgreSQL Storage
- Streamlit Dashboard
- Docker Support

## Tech Stack

Python, Kafka, PostgreSQL, FastAPI,
Streamlit, Docker, HuggingFace Transformers
